简介 ​

Lark Notice Plugin 是一个面向 Jenkins 的构建通知插件，用于将构建过程和结果发送到 Lark、飞书、钉钉、企业微信 等协作平台。

插件支持多个通知时机，包括 构建开始、构建中止、构建失败、构建成功、构建不稳定、未构建 等。

插件支持多种消息类型，包括 文本消息、图片消息、群名片消息、富文本消息、链接消息、Markdown 消息、卡片消息，以及 Lark / 飞书 场景下的 自定义 JSON 卡片消息。同时支持通过变量、模板、独立消息语言与重试策略自定义通知内容，便于在不同项目中复用。

前言 ​

在持续集成和持续交付场景中，构建结果的及时同步是团队协作的重要环节。Jenkins 作为常用的自动化构建平台，通常承担代码构建、测试执行、制品发布等任务。

仅依赖 Jenkins 页面查看构建状态，往往难以及时覆盖团队协作需求。将构建结果主动推送到协作平台，可以缩短信息传递路径，提高问题响应效率。Lark Notice Plugin 的目标就是在 Jenkins 与协作平台之间建立稳定、可配置的通知链路。

如果在使用过程中发现问题，或存在需要通过修改源码才能满足的通用需求，欢迎提交 Issue 或 Pull Request。
